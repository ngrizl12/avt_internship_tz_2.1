## POST /api/1/item - Создание объявления

| ID | Заголовок | Предусловия | Шаги | Ожидаемый результат | Комментарии |
|---|---|---|---|---|---|
| **API-POST-001** | **Создание объявления с валидными данными** | 1. Доступ к API<br>2. Уникальный sellerID | 1. Создать запрос с телом: {sellerID: int (в диапазоне 111111-999999), name: "string", price: int (0 <), statistics: {likes: int (0 <), viewCount: int (0 <), contacts: int (0 <)}}<br>2. Выполнить POST /api/1/item | 1. Код ответа : 200<br>2. Все обязательные поля присутствуют<br>3. Возвращается id товара | |
